03 / 04

✅它好在哪

01

扩大了想象力

以前大家只用 AI Agent 写代码。OpenClaw 让你意识到：Agent 可以帮你订酒店、做行程、监控股票、 写日报、管理 Notion——一切你在电脑上做的事，它都可以做。

💡 我让三个 Coding Agent（Claude Code / Codex / Gemini）各自设计网页，互相评审，投票选最佳方案——整个过程我只下了一条指令。
02

接近聊天工具，极低使用门槛

不需要打开网页、不需要登录后台——在微信/Telegram/Discord 里发条消息就行。 可以拉进群聊让它参与讨论，也可以私聊安排任务。用起来像在跟一个真人同事对话。

💡 躺在床上用手机发条语音"帮我查一下明天东京的天气和餐厅"，起床就能看到完整方案。
04 / 04

⚠️它的问题

01

巧妇难为无米之炊

Agent 再聪明，也受限于它能拿到的信息。国内生态的壁垒让很多数据拉不到：

🚧 微信公众号、小红书、大众点评、12306、淘宝……这些平台要么没有 API，要么有严格的反爬机制。 Agent 空有一身本事，但"没有手"可以伸进去。

我的解决方案是自己写了大量 Skills（相当于教它用特定的 API/工具），但这不可持续。
02

安全性问题很大

OpenClaw 默认拥有极高权限——文件系统、Shell、浏览器、消息发送全部开放。 这意味着一旦 Agent 出错或被注入恶意指令，后果不堪设想。

"This should terrify you" — Meta 超级智能团队的安全主管（Safety Director） 在使用 OpenClaw 时，Agent 失控删除了她的大量邮件，差点清空整个收件箱。 社交媒体上称这是"讽刺的巅峰"——连 AI 安全专家都没逃过。

— Fortune / MSN 报道，2026 年 3 月
🔓 Prompt Injection 风险：Agent 读取网页/邮件时可能被恶意内容劫持指令
🔓 权限过大：默认可执行任意 Shell 命令、读写任意文件
🔓 社区已有多起误操作报告：删除文件、发送错误消息、暴露隐私数据
